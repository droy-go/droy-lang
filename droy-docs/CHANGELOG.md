# Changelog

All notable changes to the Droy Programming Language will be documented in this file.

## [3.0.0] - 2026-02-27

### Added
- **SET / SETUP Naming**: Define named elements with `set=topbar=main-nav` syntax
- **TOOL Concept**: Create reusable tools with `tool=get: formatter`
- **GET_SET Pattern**: Get and set values simultaneously
- **Value Set with Operations**: Define multiple values with calculations `value-set=(a=1,b=2,c=a+b)`
- **Advanced UI Components**: Topbar, Sidebar, Header, Footer, Nav, Menu
- **Data Handling**: JSON, CSV, XML, YAML data declarations
- **Server Integration**: HTTP requests (GET, POST, PUT, DELETE, PATCH) and WebSocket
- **Group / ID / Name**: Element grouping and identification
- **Color Blending**: Advanced color operations and gradients
- **Math Functions**: sum, avg, min, max, random, round, floor, ceil, abs

### Changed
- Enhanced syntax highlighting for all new concepts
- Improved UI generator with layout components
- Updated documentation with comprehensive examples

## [2.0.0] - 2026-02-20

### Added
- Ultra-short syntax for variable declaration
- UI components: btn, img, video, icon, text, title, subtitle
- Layout components: container, grid, flex, row, col, card
- Color and background support
- Event handlers: @click, @hover, @keydown, etc.
- Animation and transition support

## [1.0.0] - 2026-02-15

### Added
- Initial release of Droy Language
- Core syntax: set, get, var, func, if, else, for, while
- Basic UI components
- Print and input functions
- Array and object support
